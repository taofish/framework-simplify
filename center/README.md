# 水平垂直居中
## 1、定宽高
- absolute（绝对定位）+margin负值
- absolute（绝对定位）+translate(-50%, -50%)
- absolute（绝对定位）+top/right/bottom/left为0+margin:auto
- flex布局
- grid布局
- 父元素table-cell+vertical-align + 子元素inline-block/margin: auto
## 不定宽高
- absolute（绝对定位）+translate(-50%, -50%)
- 父元素table-cell+vertical-align + 子元素inline-block/margin: auto
- flex布局
- grid布局
